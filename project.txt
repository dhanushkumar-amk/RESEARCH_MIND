## ResearchMind — Complete Information

---

## 1. What is ResearchMind

A production-grade AI research assistant that combines internal document knowledge with real-time external research. User uploads any document or pastes any URL — system understands it deeply, answers questions, and generates structured research reports automatically using 5 specialized AI agents.

---

## 2. Problem it Solves

| Problem | Solution |
|---|---|
| Searching multiple sources manually | 10 MCP tools search automatically |
| Reading entire documents | RAG retrieves only relevant chunks |
| LLM hallucinations | Critic agent + RAGAS scoring |
| No source grounding | Every answer has source links |
| Repeated expensive LLM calls | Semantic cache — 80% cost saved |
| Pipeline breaks in production | 10-model fallback — 99.9% uptime |

---

## 3. Core Features

1. **Document Ingestion** — PDF, Word, Excel, YouTube, websites
2. **RAG Pipeline** — Hybrid search + reranking + semantic chunking
3. **5-Agent System** — Retrieval, Research, Critic, Summary, Memory
4. **10 MCP Tools** — Tavily, Wikipedia, ArXiv, PubMed, HackerNews
5. **LLM Gateway** — LiteLLM → LLMGate, 10-model fallback
6. **Security Layer** — Guardrails AI + prompt injection defense
7. **Context Engineering** — Token budget + summary memory
8. **Evaluation** — RAGAS + LangSmith + MLflow
9. **Research Reports** — Structured output with sources
10. **Real-time Streaming** — SSE token by token response

---

## 4. 5-Agent Architecture

```
User Query
     ↓
Retrieval Agent ←→ Research Agent (parallel)
     ↓
Critic Agent — scores quality ≥ 7/10
     ↓
Summary Agent — generates report
     ↓
Memory Agent — saves context
     ↓
Answer streamed to user
```

---

## 5. RAG Pipeline Flow

```
Document uploaded
     ↓
ETL — extract text
     ↓
Semantic chunking
     ↓
HuggingFace embedding — local, free
     ↓
Store in Qdrant + AWS S3
     ↓
User asks question
     ↓
Hybrid search — BM25 + Vector
     ↓
Cross-encoder reranking
     ↓
Context budget management
     ↓
Pass to agents
```
